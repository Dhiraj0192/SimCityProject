Compilation Instructions:
-----------------------
To compile the program, use the following command:
g++ -std=c++11 *.cpp -o simcity

Running Instructions:
-------------------
1. Run the executable: ./simcity my_config.txt region1.csv
2. Follow the prompts to input:
   - Region layout file name
   - Maximum time steps
   - Refresh rate

The program will simulate city growth according to the specified rules and display results.

Bonus Implementation: No

Note: Make sure input files are in the correct CSV format as specified in the project requirements.

------------
